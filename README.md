# Configure NSQ consumers and producers based on Krakow from environment

## Environment variables

- `NSQLOOKUPD_URLS` (default to `http://localhost:4161`)
- `NSQD_TLS`        (default to `false`)
- `NSQD_HOSTS`      (default to `localhost:4150`)
- `NSQD_TLS_CERT`
- `NSQD_TLS_KEY`
- `NSQD_TLS_CACERT`
